# 1 快捷键

## 1.1  常规
- **创建多个光标**
Alt + “鼠标选择”：点击几次就创建了几个光标
Alt + Shift + i:对选中的行末添加光标
- **文件跳转**
在资源管理列表中
Ctrl + P:展示最近操作的文件，选择文件即可跳转，若选择文件后输入“:” + 行号，即可跳转到文件的相应行
Ctrl + Tab:展示目录全部文件

- **行跳转**
Ctrl + g:调入行输入框。再输入行号，即可跳转到相应行

- **查看文件中符号**
Ctrl + Shift + O：显示当前文件的所有符号
Ctrl  + T

- **跳转到定义**
Ctrl + F12:跳转到函数或字段的定义
-  **查看什么地方用到**
Shift + F12:查看被什么地方调用

## 1.2 光标移动

- **针对单词的光标移动**
Ctrl+左方向键：单词前面
Ctrl+右方向键：单词尾部

- **光标移动到行首或者行末**
Home:行首
End：行末

- **代码块的光标移动**
Ctrl + Shift + \:  花括号（代码块）之间跳转

- **跳转文档的第一行或者最后一行**
Ctrl + Home:文档首行
Ctrl + End:文档末行

## 1.3 文档选择
- **针对单词的选择**
Ctrl + Shift + 左方向键：选中当前位置到单词首部
Ctrl + Shift + 右方向键：选中当前位置到单词尾部

- **针对行的选择**
Shift + 上方向键：选中当前位置到上一行
Shift + 下方向键：选中当前位置到下一行
Shift + Home:当前位置到行首
Shift + End:当前位置到行末

## 1.4 删除操作
Ctrl +  Delete:删除当前到行末
Ctrl + Backspace:删除当前位置到单词前部

## 1.5 复制
Alt + Shift + 上方向键：当前行向上复制
Alt + Shift + 下方向键：当前行向下复制

## 1.6 移动
Alt + 上方向键：当前行上移动一行
Alt + 下方向键：当前行下移动一行

## 1.7 格式化代码
Alt + Shift + F：文档格式化

## 1.8 鼠标操作
- 单击鼠标左键，把光标移动到相应的位置。
- 双击鼠标左键，将当前光标下的单词选中。
- 连续三次按下鼠标左键，选中当前这一行代码。
- 连续四次按下鼠标左键，选中整个文档。
- 在拖拽这段文本的同时，按下 Ctrl 键，鼠标指针上会多一个加号，这时再移动鼠标或虚拟光标至我们想要的位置，然后当我们松开鼠标左键的时候，这段文本将会被复制粘贴到虚拟光标所在的位置，也就是我们既定的目标位置。移动鼠标的过程中，多按了个 Ctrl 键，操作结果就由原来的“剪切+粘贴”变为“复制+粘贴”了。
- 鼠标要拥有中键。只需按下鼠标中键，然后对着一段文档拖出一个框，在这个框中的代码就都被选中了，而且每一行被选中的代码，都拥有一个独立的光标。
-  Ctrl + 鼠标左键 ：来打开超级链接
